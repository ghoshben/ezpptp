# ezpptp

A PPTP VPN setup script for a Debian/Ubuntu VPS (KVM)

### Usage:
```bash
wget https://raw.githubusercontent.com/cwaffles/ghoshben/master/ezpptp.sh
chmod +x ezpptp.sh && ./ezpptp.sh
```
